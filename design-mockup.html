Thème Nord — tons froids bleu arctique
Structure — 1 seul écran :
  • Topbar — Logo + titre · Statut connexion (vert/rouge) · Bouton thème · Bouton file d'attente (avec compteur)
  • Barre recherche — Mots-clés ou URL YouTube/playlist
  • Dossier — Chemin actuel + bouton "Changer" → sélecteur natif
  • Résultats — Cartes horizontales : thumbnail + overlay play, badge durée, titre, chaîne, stats, boutons Télécharger + Aperçu
Queue en overlay :
  • S'ouvre via le bouton dans la topbar
  • Se ferme via ou en cliquant en dehors
  • Barre de progression + vitesse en direct
  • Pause / Reprendre / Annuler sur chaque élément : démarrer un élément en pause, mettre en pause un téléchargement actif, annuler/supprimer
  • Nouveaux téléchargements mis en "pause" par défaut si 2 sont déjà actifs
  • Statuts : en cours, en pause, en attente, terminé, erreur
Mode hors-ligne :
  • L'interface s'affiche normalement même sans connexion
  • Statut "Déconnecté" dans la topbar (rond rouge)
  • Message explicite dans la zone résultats : "Aucune connexion internet"
  • Les fonctionnalités (recherche, téléchargement) sont simplement désactivées
Fonctionnalités clés :
  • Font Awesome local — chargé depuis les fichiers locaux du projet, fonctionne sans internet
  • Pause/Reprendre les téléchargements actifs (SIGSTOP/SIGCONT)
  • Annulation de téléchargement
  • Lecteur intégré — aperçu vidéo dans une modale sans quitter l'app (embed YouTube)
  • Choix du dossier de destination natif
  • Max 2 téléchargements simultanés
